# Проект cooking_site

верстка с первого задния была шаблонизирована и используется для создания сайта ответов на сервере django
Хранится приложение в папке cooking_site, состоящая из:
- app - папка, где содержится все основные основная обработка наших страниц и создаются наши urls
- cooking_site - папка, в которой содержится основные настройки нашего сайта (settings) и views страниц
- static - папка, в которой содержатся статические шаблоны для классов css, скриптов на js и рисунков img
- templates - папка для шаблонов наших страниц. Для основ шаблонов использовались страницы из первого задания
- manage.py - программа для управления нашим приложением
- .env.example - пример параметров виртуального окружения для людей, которые в последствии могут воспользоваться сайтом
- requirements.txt - текстовый файл, который указывает, какие приложения и какой версии необходимо установить для стабильной работы приложения

## Как запустить проект на django

1. Клонируйте Репозиторий

2. Создайте и активируйте виртуальное окружение (python -m venv venv 
venv\Scripts\activate)

3. Установите зависимости pip install -r requirements.txt

4. Примените миграции (python manage.py migrate)

5. Запустите сервер (python manage.py runserver)
